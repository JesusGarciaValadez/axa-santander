Etiquetas para organizar tu información

El etiquetado es una herramienta que te servirá para “marcar” un documento y que de esta manera sea más fácil clasificarlo y localizarlo a través de palabras clave.

Con las etiquetas ahorrarás mucho tiempo en tus búsquedas.
